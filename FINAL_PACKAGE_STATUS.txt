Enable Dependabot in your GitHub repo:

1. Go to Settings → Security → Dependabot
2. Enable "Dependabot version updates"
3. Add .github/dependabot.yml:

```yaml
version: 2
updates:
  - package-ecosystem: "npm"
    directory: "/"
    schedule:
      interval: "weekly"
    open-pull-requests-limit: 10
```

This will automatically create PRs when updates are available!
