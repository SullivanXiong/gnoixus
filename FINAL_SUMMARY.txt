🚀 QUICK START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install dependencies:
   $ npm install

2. Build the extension:
   $ npm run build

3. Load in browser:
   Chrome:  chrome://extensions/ → Load unpacked → Select 'dist' folder
   Firefox: about:debugging → Load Temporary Add-on → Select manifest.v2.json

4. Start using:
   - Click extension icon to open popup
   - Toggle features on/off
   - Visit websites to see features in action!
